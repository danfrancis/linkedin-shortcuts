linkedin-shortcuts

Userscript for keyboard shortcuts on LinkedIn.com

Download this project as a .zip file Download this project as a tar.gz file

Description

This is a userscript using TamperMonkey on Google Chrome to bind Keyboard Shortcuts for LinkedIn.

Installation Steps (for Google Chrome)

  1. Add TamperMonkey to Chrome
  2. Click to install the linkedin-shortcuts userscript

Keyboard Shortcuts

Key Function
Alt + \ Help (Pop-up)
Alt + / Main Search Box
Alt + h Navigate to Homepage
Alt + p Navigate to Profile Page
Alt + i Navigate to Invitations Page
Alt + m Navigate to Messages Page
Alt + c Navigate to Contacts Page
Alt + w Navigate to Who's View My Profile Page
Alt + k Navigate to People You May Know Page
Alt + [1-5] Navigate to Search Result #
